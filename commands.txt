PM2
PM2 (Process Manager 2) is a fast and easy solution for hosting your Nuxt application on your server or VM.
To use pm2, use an ecosystem.config.cjs:
ecosystem.config.cjs

module.exports = {
  apps: [
    {
      name: 'NuxtAppName',
      port: '3000',
      exec_mode: 'cluster',
      instances: 'max',
      script: './.output/server/index.mjs'
    }
  ]
}
